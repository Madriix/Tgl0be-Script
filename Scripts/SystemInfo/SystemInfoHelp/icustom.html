Intelligent Custom Line (I know my names suck)

Brain child of cannibis and myself, I decided that the custom line just didn't cut it :) So I made something with my anger, and the Intelligent Custom Line developed.

In the SystemInfo by C}{$ setup dialog, there should be a tab called ICustom line, with several tick boxes going down. Whatever is ticked, will be parsed when /icust is used. It is parsed in order from top to bottom. I plan to put in some sort of ordering when I have time.

Remember not to select too many options otherwise you will more than likely create "line too long" error messages.

NEW IN 1.1: The option has been created so that you can make the icustom line a multiline thats intelligent and stuff. This is on by default.
