Motherboard Monitor Integration

Motherboard Monitor in SystemInfo by C}{$ now comes deactivated by default due to the amount of stupid errors it causes :). To activate it, go to the Options 1 tab and un-check "Deactivate MBM from all outputs".

Motherboard monitor is a system information tool for Windows. One feature of it is shared memory, and moo.dll takes advantage of this for fan speeds, CPU temp, and system temp.

To download Motherboard monitor and for setup instructions, go to http://mbm.livewiredev.com

To deactive MBM from all outputs in the script, go to the Options 1 tab and check the option to deactivate it.
