Commands for SystemInfo by C}{$!

/stats <---- Displays CPU, OS, Uptime, memory usage and HD info on one line
/periph <---- Displays your graphics card and sound card!
/cust <---- Allows you to create custom messages with the variables. Check the dialog setup for info!
/icust <---- Displays the new Intelligent Custom Line!
/info <---- Displays EVERYTHING on multiple lines.
/uptime <---- Displays OS, uptime and record uptime.
/osinfo <---- This shows OS Installation information such as the install directory and when it was installed
/systeminfo <---- Brings up the Dialog which displays nearly all the information in a neat dialog.
/cpuinfo <---- Displays CPU info.
/meminfo <---- Displays memory usage.
/pageinfo <---- Displays Page File Information
/vinfo <---- Displays Virtual Memory Information
/conn <---- Displays information about your current dial-up connection. (See notes in moo.dll readme)
/ni <---- Displays detailed info about your network interfaces.
/bw <---- Displays current local bandwidth usage.
/space <---- Diplays Total/free space and also the same for each partition/disk.
/space X <---- Where X is the letter of a fixed drive... will display more detailed information about X disk/partition :)
/space PATH <---- Where path is the path to a file, will display basic file information and with the -c switch, will perform a CRC and display that also, this can take a while on large files and is generally not recomended.
/space PATH <---- If the PATH is a folder, folder information is displayed :)
/netspace <---- Shows all network drives.
/display <---- Shows display information including monitor.
/client <---- Displays client version and moo.dll version.
/temp <---- See note.
/syssetup <---- Brings up Setup dialog.
/uninstallSystemInfo <---- Uninstalls SystemInfo by C}{$
/reinstallSystemInfo <---- Reloads and reinitialises SystemInfo by C}{$
/SystemInfoHelp <---- Brings up what your reading right now.
