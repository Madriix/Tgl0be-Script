Custom Line

The custom line allows you to create a completely customizable line with tokens you can add which change dynamically as time goes by.

Tokens are:

@os - Operating System

@up - Uptime

@cpu - CPU

@mem - Memory Info

@RAMBAR - RAM/Memory usage meter thing

@PAGEINFO - Page File Information

@PAGEBAR - Page File Usage Meter

@VMEMINFO - Virtual Memory Information

@VMEMBAR - Virtual Memory Usage Meter

@hd - Hard disk totals

@ni - Network Interfaces

@res - Display Info

@temp - MBM info

@mon - Monitor

@gfx - Graphics card

@snd - Sound card

To use this line, type /cust in a mIRC window.
