About SystemInfo by C}{$

SystemInfo by C}{$ started as a shitty little addon that did nothing more than display SystemInfo. Now it's still a shitty little addon that displays SystemInfo but lets you change the colours :D Not much of a history, but it's only been going 6 months. Produced quite a nice little following and all your comments are appreciated and noted.

BETA 2 CHANGES

To install beta 2 if you already have BETA 1:

  1. Download the BETA 2 archive.
  2. Extract the files into your mIRC directory or the directory where the script is.
  3. Right click the channel, go to the system info menu and go to Reload Script.

To install in general:

  1. Unload all previous versions of sysinfo. Using /unload -rs FILENAME.mrc
  2. Download the BETA 2 archive.
  3. Extract the files into your mIRC directory or the directory where the script is.
  4. Type /load -rs SystemInfo.mrc or add the directory to where you extracted it.
